{# ───────────────────────  RESTORE  ─────────────────────── #}


  
RESTORE

  

    Load a previously-downloaded backup. Accepts both .zip
    (full backup including covers) and .json (data only).
    This replaces every record currently in the library.
    The whole load runs inside a single transaction; a failure leaves
    your existing data untouched.

{# ───────────────────────  IMPORT  ─────────────────────── #}


  
IMPORT

  

    Bulk-add comics from a CSV. The wizard parses your file, lets you map
    columns, choose which sources to query, review per-row search hits,
    and commit only what you confirm. Empty rows and section-header rows
    are skipped automatically.
  

  

    Looking for ISBN/UPC/title search for a single comic? Use
    Add instead.

{# ───────────────────────  CLEANUP  ─────────────────────── #}


  
CLEANUP

  

    Sweep up orphan series — Series rows that no Comic
    points at anymore. Usually a leftover from early add-flow tweaks or
    from comics that were deleted before auto-pruning was wired in.
    Safe: removes nothing that's referenced.
  

  
  


  
FIND INCONSISTENCIES

  

    Read-only sweep that flags comics whose stored data shape disagrees
    with itself — typically the result of a wrong-pick during the import
    wizard. Each flag is a suggestion; click Review → on
    a row to land on its /repick page and pick the right
    candidate.
  

  
  




{# ───────────────────────  DANGER ZONE  ─────────────────────── #}


  
☢ DANGER ZONE

  

    Factory reset. Empties every user-data table — comics, copies,
    series, publishers, creators, characters, story arcs, tags, import
    sessions, and the upstream metadata cache. The schema stays put so
    the app keeps running; you can start adding comics again immediately.
  

  

    There is NO undo. Take a backup first if you might want this back.
