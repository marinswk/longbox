# Code of Conduct

This project adopts the
**[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. The canonical text lives on the Contributor
Covenant website; treating it as the source of truth keeps this file
short and avoids drift if the upstream is revised.

## Scope

The Covenant applies in every Longbox project space — issues, pull
requests, discussions, and any other forum where someone is acting in
the role of a project contributor or maintainer. It also applies when
representing the project in public.

## Reporting

Reports go to **marins.wk@gmail.com**. The maintainer reads every
report personally and will respond within 7 days. Reports are
confidential; reporter identity is not shared without permission.

The same address is the security contact (see
[SECURITY.md](SECURITY.md)) — both surfaces are read by the same
person, so use whichever channel feels right for the situation.

## Enforcement

The maintainer is the final authority on enforcement decisions. The
escalation ladder follows the Contributor Covenant's published
guidelines (correction → warning → temporary ban → permanent ban).

Because Longbox is a small project, most situations will resolve at
the "correction" or "warning" step via a private conversation.
